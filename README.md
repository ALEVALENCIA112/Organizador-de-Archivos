El script "organizador.py" es un archivo que permite ordenar los archivos de la carpeta de descargas en diferentes
carpetas para evitar el caos de buscar documentos importantes o temporales

Actualmente se encuentra en desarrollo por lo cual se seguiran añadiendo mejoras en cuanto a:
    Distribución de las carpetas
    Extensiones de archivos
    Ejecutable del script

Este script es desarrollado con el fin de aprender procesos de automatización a la ves que se busca maneras de facilitar
la vida diaria en el uso de la PC.
